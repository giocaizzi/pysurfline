Visualize the surf reportÔÉÅ

The data stored in a SpotForecasts object can be visualized with a plot.

[1]:

import pysurfline

[2]:

spotId = "5842041f4e65fad6a7708cfd" # anchor point
# spotId = "5842041f4e65fad6a7708890" # pipeline

spotforecasts = pysurfline.get_spot_forecasts(spotId)
spotforecasts

[2]:

<pysurfline.core.SpotForecasts at 0x113042c10>

The plot can be customized by calling the following arguments: - barLabels (bool) : height label of surf - wind (bool) : wind direction and speed

[3]:

pysurfline.plot_surf_report(
    spotforecasts,
    barLabels = True,
    wind = True,
    )
