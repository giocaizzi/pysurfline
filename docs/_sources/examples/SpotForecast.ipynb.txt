    "\n",
    "The class `SpotForecast` is an object optimized to store Surfline API Forecast data.\n",
    "\n",
    "At the base of its structure there are the `Surf`, `Wind`, `Weather`, `Swell` classes. These objects togheter form an hourly `Forecast` as per the following scheme:\n",

    "```\n",
    "class Forecast:\n",
    "    timestamp: Time\n",
    "    weather: Weather\n",
    "    wind: Wind\n",
    "    surf: Surf\n",
    "    swells: List[Swell]\n",
    "```\n",
    "\n",

    "class SpotForecast:\n",
    "    name: str\n",
    "    spotId: int\n",

    "    sunriseSunsetTimes: List[SunriseSunsetTime]\n",
    "    tides: List[Tide]\n",
    "    tideLocation: TideLocation\n",
